# Example Documentation

Welcome to the **Markdown Viewer**! This is an example documentation file to demonstrate the features.

## Getting Started

This viewer supports all standard markdown features including:

- **Bold text**
- *Italic text*
- `Inline code`
- [Links](https://example.com)
- And much more!

## Code Blocks

Here's an example of syntax highlighting:

```javascript
function greet(name) {
    console.log(`Hello, ${name}!`);
}

greet('World');
```

```python
def greet(name):
    print(f"Hello, {name}!")

greet("World")
```
